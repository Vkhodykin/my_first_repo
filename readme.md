## My first project
### Console application "Income and expense journal"

Development language: Python 3  
Application type: CLI  
Data storage: files  
Data format: JSON

Planned features:
- Adding an operation (expense or income)
- Viewing the list of operations
- Deleting an operation
- Filtering transactions by type (expense/income)
- Display general statistics for transactions
- Correct loading of data when launching the application
- Editing an operation (any of its attributes)
- Filtering transactions by category
- Displaying the balance (income minus expenses)
- Displaying statistics by category
- Search for operations in text
- Filtering transactions by period (by date or by record range)
- Sort transactions by amount or by date/order of addition
- Exporting transactions or a report to a separate file
- Separate viewing mode for "top categories" by expenses
